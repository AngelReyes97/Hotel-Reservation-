# Hotel-Reservation-

Instructions:

1. Mircosoft Visual Stuoids Installed.
2. Open Visual Studios.
3. **IMPORATNT** one extension REQUIRED
   a. Tools -> NUGet Package Manager -> Manage NUGet Packages for Solution
   b. In Browse search for FontAwesome.Sharp
   c. Install Version 5.15.4

Connect to Database in Visual Studios:
1. Download Hotel.mdf database file locate file where every was saved to.
3. Click on "Server Explorer" will be on your left under "ToolBox"
4. click on connect to database
5. click browse under database file name
6. Select Hotel.mdf and click open
7. Click OK on the add connection form   
https://github.com/AngelReyes97/Hotel-Reservation-/assets/139978507/c966b2cc-7772-40be-847c-df83cabf18d8

MAKE SURE TO CHANGE ALL CONNECTIONS ON EACH FORM:
1. Click on Hotel.mdf
2. On bottom right copy the "Connection string" copy and pasteto each form
3. These forms include double click on each Cart.cs, Edit.cs, NewReservation.cs, Staff.cs, Save Edit.cs
